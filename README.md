# mind-map-notes
Often, plain text notes just doesn't organize complext topics well enough and your folders just become a mess when trying to cretae a logical system for all you notes

And what if a concept builds off aother central idea, but both should be its own object?

> This is the core idea of this project. A note taking tool that allows you to create
a mind map of nodes, linked based off your own conections
